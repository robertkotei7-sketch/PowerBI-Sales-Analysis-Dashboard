# PowerBI-Sales-Analysis-Dashboard
Power BI dashboard analysing sales trends, profit by category, Top 3 selling products, and map-based state performance with interactive slicers for region, segment, and year.

This project demonstrates an end-to-end Power BI analysis using the dataset provided. The dashboard focuses on sales performance, profitability, customer insights, and geographic trends.

## Visualisations Implemented

### 1. Sales Trend Analysis
- **Line Chart** showing total sales over time
- Axis: Order Date  
- Value: SUM of Sales  
- Used to identify growth patterns and seasonality

### 2. Profitability Analysis
- **DONUT Chart** analysing Profit by Category
- Helps compare performance across:
  - Furniture
  - Office Supplies
  - Technology

### 3. Top Product Performance
- **Table Visual** displaying the **Top 3 Selling Products**
- Products ranked dynamically using DAX measures
- Used TOPN and RANKX logic to highlight highest revenue-generating items

### 4. Geographic Distribution
- **Map Visual** showing Sales by State
- Identifies regional sales hotspots across the United States

### 5. Interactive Filtering
- Fully functional slicers added for:
  - **Region**
  - **Customer Segment**
  - **Year**
  - **Category**
- Enables drill-down and flexible analysis for different perspectives
